高德地图#import < MAMapKit/MAMapKit.h > file not found的解决办法

最近在看一个程序,但是运行的时候却报了一个错

#import < MAMapKit/MAMapKit.h > file not found .

仔细一看是引用了一个高德地图的SDK.
那么简单了,我看一下有没有这个sdk的库文件,没有添加上就是了.

但是仔细一看发现,这个sdk已经存在了.然后我又猜想是不是searchPath没有设置对.

好吧,看过之后,发现searchPath也没有设置错.最后无奈了,只能去这个包里面看一下什么问题了.

查看包内容后发现是如下的情况,很多的快捷方式,而且都已经失效了,可能是在工程拷贝的时候失效了.

那么解决的办法就来了.我直接versions里面的文件夹里,把对应的三个文件拷贝出来,删除这些没用的快捷方式.

这个时候,再运行,发现OK了.
